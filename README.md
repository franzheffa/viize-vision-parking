# Viize Vision Parking

VIIZE VISION — L’IA au service du stationnement urbain.

## Features (MVP)
- Interface Conducteur : réserver une place (temps réel)
- Interface Manager : tableau de bord IA (occupation + revenus)
- API Routes (Next.js App Router) : lots, reservations, pricing quote, payouts simulation, edge events
- Prisma schema + seed
- Déploiement : Cloud Run scripts + manifests GKE
